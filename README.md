# Zcat - Генератор карточек для каталогов и выставок

Веб-приложение для создания карточек с описаниями произведений искусства на основе загруженных изображений с использованием AI.

## Технологический стек

- **React 18** - библиотека для создания пользовательского интерфейса
- **Vite** - сборщик и dev-сервер
- **CSS** - стилизация

## Установка и запуск

### Требования
- Node.js (версия 18 или выше)
- npm или yarn

### Установка зависимостей

```powershell
npm install
```

### Запуск в режиме разработки

```powershell
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для продакшена

```powershell
npm run build
```

### Просмотр продакшен-сборки

```powershell
npm run preview
```

## Структура проекта

```
Zcat/
├── src/
│   ├── components/     # React компоненты
│   ├── styles/         # CSS стили
│   ├── utils/          # Утилиты и вспомогательные функции
│   ├── assets/         # Статические ресурсы (изображения и т.д.)
│   ├── App.jsx         # Главный компонент приложения

├── index.html          # HTML шаблон
├── vite.config.js      # Конфигурация Vite
└── package.json        # Зависимости проекта
```
